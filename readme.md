## 简介
本项目是shine_xcx的管理后台，主要为了自己练手用的。

## 技术栈
+ webpack@^2.0.0 这里踩了坑，事实证明跟着网上教程，install最新的webpack是不切实际的。
+ webpack-dev-server@2.0.0 踩坑记同上
+ react

## 开始吧
+ step1：node版本大于4.0.0
+ step2：

`npm install webpack@^2.0.0 webpack-cli@^2.0.0 webpack-dev-server@^2.0.0 babel -g`

+ step3:

`npm install`

+ step4:

`npm start`
